# use-scroll-sense

A highly efficient, zero dependency React hook for smooth scrolling and Intersection Observer scroll spying. Perfect for modern portfolios, landing pages, and complex navigation systems.

## Installation

```bash
npm install use-scroll-sense

Features
useScrollSpy: Tracks which section of the page is currently in the viewport using the native IntersectionObserver API. Highly optimized to prevent unnecessary re-renders.

useSmoothScroll: Programmatically scrolls to specific elements on the page safely, without SSR (Server-Side Rendering) window errors.

TypeScript Support: Written entirely in TypeScript with full type definitions exported.
